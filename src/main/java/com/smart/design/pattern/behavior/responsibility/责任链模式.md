# 责任链模式(Chain Of Responsibility)
责任链模式(Chain Of Responsibility),又叫责任链模式，为请求创建了一个接收者对象的链。
使多个对象都有机会处理请求，从而避免请求的发送者和接收者之间的耦合关系。将这个对象连接成一条链，并沿着这条链传递该请求，直到有一个对象处理它为止。
